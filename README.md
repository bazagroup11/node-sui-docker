# Node Sui Docker.

## Команды.

- сборка образа:
  ```bash
    yarn build
  ```

- просмотр образов:
  ```bash
    yarn images
  ```

- удаление образов:
  ```bash
    yarn images:remove
  ```

- просмотр контейнеров:
  ```bash
    containers
  ```

- остановка контейнеров:
  ```bash
    containers:stop
  ```

- удаление контейнеров:
  ```bash
    containers:remove
  ```
